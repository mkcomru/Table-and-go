# Тестовые данные для TableAndGo

Этот каталог содержит скрипты для создания тестовых данных для проекта TableAndGo.

## Содержимое

1. `create_tables.py` - скрипт для создания необходимых таблиц в SQLite
2. `fill_data.py` - скрипт для заполнения таблиц данными из файлов `data_restaurants.txt` и `deta_bars.txt`

## Требования

Для корректной работы скриптов необходимо:

1. Настроенный проект Django
2. Файлы с данными:
   - `data_restaurants.txt` - данные о ресторанах
   - `deta_bars.txt` - данные о барах

Файлы с данными должны находиться в корневой директории проекта.

## Использование

### Создание таблиц

```bash
cd tableandgo
python test_data/create_tables.py
```

### Заполнение таблиц данными

```bash
cd tableandgo
python test_data/fill_data.py
```

## Формат данных

Файлы данных должны соответствовать следующему формату:

```
название: Название заведения
тип заведения: Ресторан/Бар
описание: Описание заведения
email: example@email.com
сайт: https://example.com
тип кухни: Тип1, Тип2, Тип3
филиал 1: 
название: Название филиала
основной: да/нет
адрес: Адрес филиала
район: Название района
телефон: Номер телефона
средний чек: Сумма
часы работы: понедельник(11.00-24.00), вторник(11.00-24.00),...
```

## Примечания

1. Если заведение уже существует в базе данных (проверка по имени), оно не будет создано повторно
2. Часы работы могут быть определены как:
   - Обычное время: `день(часы.минуты-часы.минуты)`
   - Выходной: `день(выходной)`
   - Круглосуточно: `день(круглосуточно)`
